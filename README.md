# Coders of Bangalore

A Python-based data extraction and preprocessing project that converts unstructured social media profile data into a clean and structured JSON dataset.

# Features
- Data Extraction
- Data Cleaning
- JSON Data Processing
- Profile Information Parsing
- Structured Dataset Generation

# Tech Stack
- Python
- JSON
- Jupyter Notebook

# Output
Cleaned dataset containing:
- Username
- Posts Count
- Followers
- Following
- Page Category
- Bio
